# Messi vs Ronaldo - Comparison Website

## Overview

A comprehensive, interactive web application dedicated to exploring and comparing two of football's greatest legends: **Lionel Messi** and **Cristiano Ronaldo**. This website provides detailed information, statistics, career highlights, and stunning photo galleries of both players in an engaging, responsive interface.

## Live Demo

🌐 **Visit the live website:** [https://ahammad204.github.io/messivsronaldo/](https://ahammad204.github.io/messivsronaldo/)

## Features

### 📱 **Responsive Design**
- Mobile-first responsive layout using Bootstrap 5
- Optimized for desktop, tablet, and mobile devices
- Smooth and intuitive user interface

### 🏠 **Home Page**
- Dynamic image carousel showcasing football moments
- Quick introduction to both players
- Easy navigation to detailed player pages
- Visual comparison overview

### 👤 **Player Pages**
- **Messi Profile** - Comprehensive biography and career information
  - Personal details (birth date, height, family)
  - Career achievements and milestones
  - Club history (FC Barcelona, Paris Saint-Germain)
  - International career highlights
  
- **Ronaldo Profile** - Detailed information about Cristiano Ronaldo
  - Early life and career development
  - Current team and statistics
  - Personal background and journey
  - Career timeline

### 🖼️ **Photo Galleries**
- **Messi Photo Album** - Curated collection of iconic Messi photos
- **Ronaldo Photo Album** - Stunning Ronaldo image collection
- Interactive image galley with hover effects
- High-quality images with responsive layout

### ⚡ **Interactive Elements**
- Responsive navigation bar for easy page access
- Image zoom effects on hover
- Voting functionality for fan engagement
- Clickable elements with visual feedback

## Installation & Local Development

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/ahammad204/messivsronaldo.git

# Navigate to project directory
cd messivsronaldo

# Open in a web server or directly in browser
# For Python 3
python -m http.server 8000

# For Python 2
python -m SimpleHTTPServer 8000

# Then visit: http://localhost:8000
```
